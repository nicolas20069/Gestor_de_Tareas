Detalles importantes acerca de esta aplicación

  • Este gestor de tareas fue desarrollado como proyecto final del curso de Angular.
  • Permite agregar, eliminar, listar y limpiar tareas.
  • Las tareas pueden marcarse como completadas y desmarcarse dinámicamente.
  • La aplicación utiliza localStorage para mantener la persistencia de datos incluso al recargar la página.
  • Incluye validaciones para evitar tareas vacías o con longitud incorrecta.
  • Se implementó un servicio inyectable para centralizar la lógica de almacenamiento y mantener la arquitectura modular.
  • Se emplea Angular Forms para la captura de datos y un Pipe personalizado para capitalizar las descripciones.
  • La interfaz está diseñada con SCSS responsivo, adaptándose a dispositivos móviles.
  • Desarrollado por: Nicolas Melo Yandún.
  • Versión: 1.0.0
  • Fecha de lanzamiento: 15 de Abril de 2026
  • Tecnologías utilizadas: Angular, TypeScript, HTML, SCSS, Angular Router, Angular Forms, localStorage, Pipes personalizados.
  • Arquitectura: Componentes standalone y servicios inyectables con persistencia automática.
